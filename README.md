# Undertale Doggo Boss Fight :dog2:

Hey, this is my first project for CS50x. Since we had to build something we liked, I made this little boss fight based on one of my favorite games using Scratch.

You can access this game through the link: https://scratch.mit.edu/projects/652452304/

## Instructions: :keyboard:

**Heart moviment:** Use keyboard arrows

**Choosing options:** Type according to the option

**Waking up doggo:** Left click

## Objective: :dog:

Survive doggo´s cute attacks and pet him to become his friend.

# Luta contra o catiorro de Undertale :dog2:

Esse é o meu primeiro projeto para o CS50x. Já que os alunos tinham que construir algo que gostam, eu fiz essa boss fight com o doguinho de undertale usando Scratch.

Você pode acessar o jogo através desse link: https://scratch.mit.edu/projects/652452304/



## Instruções::keyboard:

**Movimentar o coração:** Use as setas do teclado

**Escolher opções:** Digite o nome da opção desejada

**Acordar o cachorro:** Clique esquerdo

## Objetivo: :dog:

Sobreviva aos ataques fofos do cachorro e faça carinho nele para vocês virarem amigos.
